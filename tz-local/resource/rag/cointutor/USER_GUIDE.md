## 1. 플랫폼 개요

### 1.1 CoinTutor란?

CoinTutor는 **암호화폐 투자 정보**를 제공하는 **구독 기반 튜터링 플랫폼**입니다.

- **일반 사용자**: 튜터를 구독하고, **학습 콘텐츠**(스터디·시험·문제)와 **코인 추천**을 이용합니다.
- **튜터**: 투자 전문가로 등록해 **학습 콘텐츠**를 만들고 **코인 추천**을 제공하며, 구독료 수익을 얻습니다.
- **통화**: 모든 결제·충전·출금·구독료·수익은 **USDT**(테더)로만 처리됩니다. (USD, KRW 등 다른 통화 표기 없음)

### 1.2 서비스 구성

| 구분 | 내용 |
|------|------|
| **학습** | 스터디, 시험, 보이스 인터뷰, 문제 풀이 (튜터별 학습 콘텐츠) |
| **추천** | 튜터의 코인 추천 (목표가, 목표일, 분석 등) — **회원 이상**만 조회 가능 |
| **구독** | 튜터별 월 구독료를 USDT로 결제해 해당 튜터의 학습·추천 이용 |
| **지갑** | USDT 충전(입금), 출금, 잔액 조회 |
| **추천인(리퍼럴)** | 링크 공유 시 구독 발생 시 수수료(커미션) 지급 |

### 1.3 멤버십 레벨 (일반 사용자)

| 레벨 | 조건 | 학습(스터디/시험 등) | 코인 추천 |
|------|------|----------------------|-----------|
| **일반회원** | 회원가입만 | ✅ 이용 가능 | ❌ 불가 |
| **준회원** | "멤버되기" 클릭, USDT 결제 전 | ✅ 이용 가능 | ❌ 불가 |
| **회원** | USDT 결제 완료 | ✅ 이용 가능 | ✅ 이용 가능 |

- **학습**: 일반회원·준회원·회원 모두 이용 가능 (튜터가 만든 스터디/시험/문제 등).
- **코인 추천**: **회원**(구독 결제 완료)부터 조회 가능. 준회원은 결제 완료 후 회원으로 전환되면 이용 가능.

## 2. 통화 및 결제

### 2.1 사용 통화: USDT만 사용

- **표시·결제·정산·출금**: 모든 금액은 **USDT**(테더) 단위로만 표시·처리됩니다.
- **표기 예**: `29.99 USDT`, `1000 USDT` (USD $, KRW ₩ 기호 사용하지 않음)
- **이유**: 암호화폐 결제 정책·규제 대응을 위해 단일 스테이블코인(USDT)으로 통일합니다.

### 2.2 구독·결제 흐름 (일반 사용자)

1. 튜터 선택 후 **구독하기** → 구독료(USDT/월) 확인
2. **지갑 잔액**으로 결제 또는 **USDT 입금** 후 결제
3. 결제 완료 시 **회원** 전환 → 해당 튜터의 **코인 추천** 조회 가능
4. 구독 갱신: 설정에 따라 자동 갱신 또는 만료 시 재결제

### 2.3 지갑 (USDT)

- **충전(입금)**  
  - TRC20(Tron) 권장: 수수료 상대적으로 낮음, 처리 1–3분  
  - ERC20(Ethereum) 선택 가능: 수수료 높음, 5–15분  
  - 플랫폼이 안내하는 입금 주소로 USDT 송금 후 확인되면 잔액 반영

- **출금**  
  - 지갑에서 출금 요청 → 관리자 승인 후 블록체인 전송  
  - **출금 수수료**: 사용자는 USDT만 받고, 네트워크 수수료(가스비)는 플랫폼이 부담하는 구조일 수 있음 (운영 정책 확인)

### 2.4 수수료 개요 (사용자 관점)

- **구독료**: 튜터가 정한 월 구독료(USDT)를 그대로 결제.
- **입금**: 블록체인 네트워크 수수료(TRC20/ERC20 등)는 사용자가 거래소·지갑에서 부담하는 부분과 동일.
- **출금**: 플랫폼 정책에 따라 무료 또는 일부 수수료 적용 (안내 문구 확인).

## 3. 일반 사용자 가이드

### 3.1 회원가입·로그인

- **소셜 로그인**: Google, Apple, Facebook 등으로 가입·로그인
- **이메일 인증**: 가입 후 이메일 인증 완료 시 모든 기능 이용 권장
- **프로필**: 언어(ko, en, ja, zh, es, fr), 타임존 설정 가능. 모든 시간은 UTC 저장, 화면에서는 사용자 타임존으로 표시

### 3.2 학습 (Learning)

학습은 **로그인한 사용자**(일반회원·준회원·회원) 모두 이용할 수 있습니다.

| 메뉴/경로 | 설명 |
|-----------|------|
| **학습 홈** (`/learning`) | 학습 메인, 진입점 |
| **스터디** (`/learning/studies`) | 튜터가 만든 스터디 목록·상세·참여 |
| **시험** (`/learning/exams`) | 튜터가 만든 시험 목록·응시 |
| **시험 응시** (`/learning/take-exam/:id`) | 선택한 시험 풀기, 제출 |
| **시험 결과** | 내 시험 결과·정답 확인 |
| **보이스 인터뷰** | 시험을 음성으로 답하는 보이스 인터뷰 시험·결과 목록 |
| **문제** (`/learning/questions`) | 문제 목록·풀이 |
| **텍스트→문제** (`/learning/text-to-questions`) | 텍스트로부터 문제 생성 (권한 있는 경우) |

- **멤버되기**: 학습 화면에서 **멤버되기**를 누르면 **준회원**이 되고, USDT 결제 완료 시 **회원**이 되어 해당 튜터의 **코인 추천**도 볼 수 있습니다.
- 학습 모드(비한국어·비한국 등)에서는 결제 금액·지갑 잔액 등은 숨기고, "멤버되기" 후 결제 절차는 별도 화면에서 진행할 수 있습니다.

### 3.3 튜터 검색·구독

| 메뉴/경로 | 설명 |
|-----------|------|
| **튜터 목록** (`/tutors`) | 튜터 검색·필터·정렬 |
| **튜터 랭킹** (`/tutors/ranking`) | 등급·온도·성공률 등 기준 랭킹 |
| **튜터 상세** (`/tutors/:id`) | 프로필, 성공률, 구독료(USDT/월), **구독하기** |
| **내 튜터** (`/my-tutors`) | 구독 중인 튜터 목록 |
| **성과** (`/performance`) | 구독한 튜터들의 추천 성과 요약 |

- 구독 시 **구독료(USDT/월)** 확인 후 지갑 잔액으로 결제하거나 USDT를 충전한 뒤 결제합니다.
- 구독 해지 시 만료일까지는 학습·추천 계속 이용 가능하고, 만료 후에는 조회 불가.

### 3.4 코인 추천 조회

- **권한**: **회원**(해당 튜터 구독 결제 완료)만 구독한 튜터의 코인 추천을 볼 수 있습니다.
- **내용**: 코인명, 현재가·목표가, 목표일, 상승률, 분석·근거, 상태(진행 중/달성/실패) 등
- **알림**: 목표 달성 시 알림(푸시·이메일 등) 수신 가능 (설정에 따름)

### 3.5 구독·결제 내역

- **구독 관리** (`/subscriptions`): 구독 중인 튜터, 만료일, 해지
- **결제 내역** (`/billing`): USDT 결제 내역 (구독료 등)

### 3.6 지갑

- **지갑** (`/wallet`): USDT 잔액, 충전(입금 주소·네트워크 선택), 출금 요청
- 충전은 TRC20 권장, ERC20 선택 가능. 출금은 플랫폼 정책·승인 절차에 따릅니다.

### 3.7 추천인(리퍼럴)

- **내 수익** (`/referral/my-earnings`): 내 추천인 링크로 가입·구독 시 발생한 **커미션(수익)** 조회
- 추천인 링크로 들어온 사용자가 구독하면, 정책에 따라 일정 비율(예: 1%)이 USDT로 지급될 수 있습니다. (정책·비율은 앱 내 안내 참고)

### 3.8 알림

- **알림** (`/notifications`): 새 추천, 목표 달성, 구독 만료 예정 등
- 프로필에서 푸시·이메일 수신 여부 설정 가능

## 4. 튜터 가이드

### 4.1 통화·수수료 (튜터)

- **구독료**: 튜터가 정한 **월 구독료는 USDT**로만 설정·표시·결제됩니다.
- **플랫폼 수수료**  
  - **기본: 30%** — 구독료의 30%는 플랫폼 수수료로 차감됩니다.  
  - **마케팅 비용**: 마케팅이 포함되면 **추가 마케팅 비용**이 별도 청구될 수 있습니다.  
  - 등급별 수수료율·마케팅 비용 상세는 **튜터 페이지(대시보드 등)**에서 확인할 수 있습니다.
- **튜터 수익**: 구독료(USDT) − 플랫폼 수수료(30% 등) − 마케팅 비용(해당 시) = 실제 수령액(USDT).

### 4.2 튜터 신청

| 항목 | 설명 |
|------|------|
| **경로** | 프로필 메뉴 → **튜터 되기** → `/tutor/apply` |
| **입력** | 닉네임, 프로필 이미지, 자기소개, 투자 경력, 월 구독료(USDT), 포트폴리오 링크 등 |
| **약관** | 튜터 이용약관, **플랫폼 수수료(30% 기본, 마케팅 시 추가 비용)** 안내 동의 |
| **승인** | 관리자 검토 후 승인 시 튜터 권한 부여, Bronze 등급부터 시작 |

신청 시 화면에 표시되는 문구 예:
- "플랫폼 수수료 30% 기본이며, 마케팅이 포함되면 추가 마케팅비용이 청구됩니다. 등급별 수수료는 튜터 페이지에서 확인할 수 있습니다."
- "플랫폼 수수료(30% 기본, 마케팅 시 추가 비용 청구)를 이해하고 동의합니다"

### 4.3 튜터 대시보드

| 메뉴/경로 | 설명 |
|-----------|------|
| **대시보드** (`/tutor/dashboard`) | 구독자 수, 월 수익(USDT), 추천 수, 성공률, 등급·온도 요약 |
| **추천 관리** (`/tutor/recommendations`) | 코인 추천 목록·등록·수정·삭제 |
| **추천 작성** (`/tutor/recommendations/create`) | 새 코인 추천 등록 (코인, 목표가, 목표일, 분석 등) |
| **구독자 관리** (`/tutor/subscribers`) | 구독자 목록·상태 |
| **활동 이력** (`/tutor/activity-history`) | 추천·구독 등 활동 로그 |
| **추천인(리퍼럴)** (`/tutor/referral`) | 추천인 링크·지급 수수료(커미션) 내역 |

- **수익 표시**: 모두 **USDT** 단위. 플랫폼 수수료 차감 후 금액이 표시될 수 있습니다.

### 4.4 코인 추천 등록·관리

- **등록**: 코인 선택, 현재가(자동 반영 가능), 목표가, 목표일, 상승률, 분석(다국어) 입력
- **등급별 조건**: 등급에 따라 **최소 보장 기간·최소 목표 상승률**이 다름 (Bronze 7일/5% ~ Diamond 45일/25% 등). 조건 미충족 시 등록 제한
- **상태**: 진행 중(pending), 달성(achieved), 실패(failed). 목표가 도달 시 시스템이 자동으로 달성 처리할 수 있음
- **월 최소 추천 수**: Silver 이상 등급은 월별 최소 추천 개수 미달 시 실패 1건 처리 등 규칙이 있을 수 있음 (대시보드·안내 참고)

### 4.5 등급·온도

- **등급**: Bronze → Silver → Gold → Platinum → Diamond. 성공률·최소 추천 수 등 조건 충족 시 승급, 실패 누적 시 강등 가능
- **온도**: 인기도·활동도 지표. 추천·구독자 활동에 따라 변동
- 등급별 수수료율·혜택(노출·할인 등)은 튜터 전용 페이지에서 확인

### 4.6 구독자·수익·정산

- **구독자 관리**: 구독 중인 사용자 목록, 구독 시작일·만료일
- **수익**: 구독료(USDT) 기준, 플랫폼 수수료(30% 등) 차감 후 금액
- **정산·출금**: 플랫폼 정책에 따라 일정 금액 이상 또는 주기적으로 USDT로 정산·출금 신청 가능 (관리자 승인·출금 수수료 정책 확인)

### 4.7 학습 콘텐츠 (튜터)

- 튜터는 **스터디·시험·보이스 인터뷰·문제**를 생성·관리할 수 있습니다.
- 학습 경로: 학습 메뉴 내 스터디/시험/문제 관리 화면에서 생성·수정·삭제

### 4.8 추천인(리퍼럴) — 튜터

- **추천인 관리** (`/tutor/referral`): 내 추천인 링크, 링크로 유입된 구독 수, **지급 수수료(커미션)** 내역
- 추천인으로 지급되는 커미션은 정책에 따라 USDT로 집계·지급됩니다.
